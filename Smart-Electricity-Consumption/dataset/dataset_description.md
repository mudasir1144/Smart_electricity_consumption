# Smart Electricity Consumption Dataset


## 1. Dataset Overview:
The smart electricity consumption dataset is an original dataset created specificially for the machine learning project **Smart Electricity Consumption & Energy optimization System** unlike publically available datasets, this dataset was entirely generated by simulating realistic househould electricity usage patterns.The objective was to create a balanced and meaningful dataset suitable for training regression based ML model capable of predicting dailt electricity consumption. The dataset represents residentials households with different house tyoes , family sizes , appliance usage pattern , eather conditions and life style behaviour.

---

## 2. Dataset Collection Method

This dataset was **no collected from Kaggle , UCI Repo , Github , or any other public source..**
Instead , it was generated using a custom python script ('dataset_generation.py') developed specifically for this project.

The dataset generation process involved:

- Randomized household characteristics
- Realistic appliance usage
- Seasonal variation
- Lifestyle Factors
- Controlled probability distribution
- Rule based electricity consumption calculation
- Small random noise to simulate real word variation

## 5. Target Variable

The machine learning model predicts:
**Daily_Electricity_Consumption (KWH)***

This value represents the estimated total electricity consumed by a housejhold during one day.
The target value was generated using a weighted combination of appliance usage  , enviromental conditions, family characterisitics , and random noise to simulate realistic electricity consumption.

---

## 6. Data Generation Logic

Electricity consumption was calculated using a rule based mathematical model.

The following factors contributed to the final consumption value:

- Base household electricity load
- Air conditioner usage
- Fan usage
- Refrigerator usage
- Washing machine usage
- Water motor usage
- Lighting usage
- Number of family members
- Work from home effect
- Outdoor temprature
- Solar panel offset
- Random noise
